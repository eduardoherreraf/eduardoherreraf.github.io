Bienvenido

Este es un ejemplo de layout con sidebar de ancho fijo y contenido flexible usando Flexbox. El sidebar mantiene sus 250px de ancho mientras que el contenido se expande para ocupar el resto del espacio disponible.

Características

✓ Sidebar con ancho fijo de 250px
✓ Contenido flexible que se adapta al espacio restante
✓ Diseño responsivo y moderno
✓ Scroll independiente en cada sección

Flexbox

El contenedor principal usa display: flex para crear el layout. El sidebar tiene width: 250px y flex-shrink: 0 para mantener su ancho fijo, mientras que el contenido usa flex: 1 para ocupar todo el espacio restante.
